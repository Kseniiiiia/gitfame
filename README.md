# GitFame

Утилита командной строки для анализа статистики авторов в Git-репозиториях. Получайте детальную информацию о вкладе разработчиков: количество строк, коммитов и файлов.

## Возможности

- **Комплексная статистика**: Подсчет строк, коммитов и файлов для каждого автора
- **Гибкая фильтрация**: Фильтрация файлов по расширениям, glob-паттернам и языкам
- **Несколько форматов вывода**: Табличный, CSV, JSON и JSON-lines
- **Умная сортировка**: Сортировка по строкам, коммитам или файлам с правильными fallback-ами
- **Интеграция с Git**: Использует `git blame --porcelain` для точного определения авторства строк
- **Обработка пустых файлов**: Корректная обработка пустых файлов через `git log`
